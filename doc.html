New Video Player HTML5


Vamos começar

Primeiro de tudo vamos nos acostumar com a tag de vídeo HTML5. Para criar um novo elemento de vídeo é só usar a tag de vídeo. É melhor usar vários tipos de vídeo, uma vez que nem todos os vídeos são suportados por todos os navegadores. Usando apenas .webm e .mp4 vai cobrir principalmente a cada tipo de vídeo embora. Miro fornece um método simples para converter arquivos para .webm.
